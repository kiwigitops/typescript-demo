# Learning Path

This repo is meant to be worked through in order. Each chapter is small enough to run, break, and fix in one sitting.

| Step | Topic | Command |
| --- | --- | --- |
| 01 | Hello world and typed functions | `pnpm lesson:01` |
| 02 | Values, literals, arrays, tuples | `pnpm lesson:02` |
| 03 | Functions and object types | `pnpm lesson:03` |
| 04 | Unions and narrowing | `pnpm lesson:04` |
| 05 | Generics | `pnpm lesson:05` |
| 06 | Async and result types | `pnpm lesson:06` |
| 07 | API client validation | `pnpm lesson:07` |
| 08 | Frontend state | `pnpm lesson:08` |
| 09 | Node API | `pnpm api` |

## Rhythm

1. Run the chapter.
2. Read the source.
3. Change one type or value.
4. Run `pnpm typecheck`.
5. Fix the compiler error deliberately.
