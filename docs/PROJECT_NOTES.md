# Project Notes

## Teaching Goals

- Keep every lesson runnable.
- Prefer realistic examples over abstract type puzzles.
- Use shared domain types once the repo reaches API and frontend examples.
- Keep runtime validation visible at program boundaries.
- Show tests as a complement to types, not a replacement for them.
